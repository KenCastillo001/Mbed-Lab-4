Mbed Lab 4 - Generación de sonido con PWM y control analógico

Práctica de sistemas embebidos con el framework Mbed sobre microcontrolador ARM.

Descripción

Generación de una señal de audio tipo diente de sierra utilizando salida PWM en un altavoz, con control dinámico de frecuencia y volumen mediante dos potenciómetros.

Hardware

Microcontrolador ARM compatible con Mbed (NUCLEO / LPC1768)
1 altavoz conectado a p21
2 potenciómetros conectados a:

p15 (AIN1)
p16 (AIN2)
Conceptos aplicados

PwmOut
AnalogIn
Modulación por ancho de pulso (PWM)
Control de frecuencia y amplitud
Conversión analógica-digital (ADC)

Plataforma

Desarrollado y simulado en Mbed Simulator
